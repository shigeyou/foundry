# 商船三井グループ_DXの取り組み.pdf

## 1. ドキュメントの目的・概要（1-2文）
商船三井グループが、経営計画「BLUE ACTION 2035」に基づき、環境・安全・非海運事業拡大・経営管理高度化・人財・ICT基盤を対象にDXを全社横断で推進する方針、体制、ロードマップ、主要施策とKPIを整理した資料です。2025年度までの「徹底的なデジタル化」と、2026年度以降の「企業価値変革」を段階的に進める構想を示しています。

## 2. 主要なポイント（箇条書き5-10項目）
- DXは「BLUE ACTION 2035」のサステナビリティ課題（マテリアリティ）の一つ（DX＝イノベーション）として位置づけられ、環境・安全を中心に“with DX”で推進。
- DXビジョンは「人・安全・社会・新たな領域」の4視点で、デジタルと共創により新領域（Blue Ocean）を切り拓くことを掲げる。
- 推進体制はCEO主導。2023年11月に技術革新本部とDX共創ユニットを統合し「技術・デジタル戦略本部」を新設、技術×デジタルを一体運営。
- ロードマップは、～2025年度「Fully Digitized/Digitalized（徹底的デジタル化と業務・組織最適化）」、2026年度以降「Transformed（企業価値・社会的存在意義の変容）」。
- 環境領域では、風力推進「ウインドチャレンジャー」、運航最適化（DarWIN/FOCUS、AI航路・速度最適化、船底汚損解析）などでGHG削減と燃費改善を狙う。
- 安全領域では、燃料油・潤滑油管理（BUNKER HUB）、AIカメラ火災検知（Captain’s Eye）、安全運航支援センター（SOSC）等で事故予防・早期対応を高度化。
- 船上業務のデジタル化（Noon Report、ログブック、整備・点検、在庫等）を進め、形式業務の負荷低減と安全業務へのシフトを図る。生成AI活用も視野。
- サイバーセキュリティは規程整備（MOL Cyber Security Standard等）とCSIRT運用、教育、船内ネットワーク標準化で強化（規制動向IACS E26/E27対応）。
- 非海運では、FSRU（LNG社会インフラ）や洋上風力O&Mなど“社会インフラ事業”拡大をDXで支え、デジタルプラットフォーム化・新モデル創出を狙う。
- 経営管理はIFRS導入と統合データ基盤（MOL PEARL）で、財務・非財務の可視化（経営ダッシュボード、GHG可視化、傭船ガバナンス等）を高度化。
- 人財面はタレントマネジメント（SAP SuccessFactors）とDX人財育成（チェンジリーダー／DXスペシャリスト、生成AIリテラシー）を両輪で推進。
- ICTはグローバル共通のデジタルプラットフォーム、SaaS最大活用、データガバナンス整備で、コラボ・生産性・統制・セキュリティを底上げ。

## 3. 具体的な数字・事実（あれば箇条書き）
- DXロードマップKPI（価値創造業務・安全業務への転換率：業務時間削減率ベース）  
  - 2025年度：10%／2030年度：20%／2035年度：30%
- チェンジリーダー（Level3完了者累計）  
  - 2023年度：30名、2024年度：75名、2025年度：120名
- 環境KPI  
  - 2030年までにGHG排出総量23%削減（2019年比、Scope1+2、連結）  
  - 2035年までに輸送GHG排出原単位45%削減（2019年比、外航自社運航船、Scope1+3一部）
- 環境投資（2023–2025年度累計）  
  - 計画：6,500億円、意思決定ベース実績：6,580億円（うち23年度実投資2,595億円）
- ウインドチャレンジャー  
  - 竣工：2隻（第1船2022年秋、第2船2024年夏）  
  - 燃料/GHG削減：1航海平均5～8%、最大17%/日  
  - 搭載計画：2030年25隻、2035年80隻（マイルストーン）
- DarWINプロジェクト  
  - 2025年までに約100億円投資、燃費効率を2019年比で5%改善目標（※資料では前倒し達成と記載）
- Wayfinder（AI最適速度・航路）  
  - トライアル（運航船40隻）で燃費効率・GHG削減とも約6%、海況予測精度50%向上、船長満足度80%
- BUNKER HUB  
  - 運航船約800隻から年約7,000本の燃料/潤滑油サンプルを分析、月間アクセス数3.7万件以上
- Captain’s Eye（AI火災検知）  
  - 2024年以降竣工予定の8隻を含む計10隻に搭載計画、世界初のClassNK認証取得
- 安全の先行指標（2019→2023）  
  - LTIF：0.38→0.26、運航停止平均時間：39.58→24.46（時間/隻・年）、運航停止発生率：0.26→0.40
- サイバー/情報セキュリティ  
  - 重大ICTインシデント件数：直近3年0件  
  - e-learning受講率：92.8%→93.9%→96.2%（2021→2023）
- 非海運ポートフォリオ（利益構成の計画）  
  - 2022年度実績：非海運25%／海運75% → 2035年度計画：非海運40%／海運60%
- タレントマネジメント（SuccessFactors）  
  - 2024年4月稼働、約8,000名の人財情報を取り込み（順次拡大）
- 生成AI活用人財育成（ハンズオン）  
  - 2024年8–10月：155名受講、98%が「業務で使えそう/挑戦したい」と回答
- CVC（MOL PLUS）  
  - 投資枠40億円、投資総件数：2021年4件→2024年25件、総投資額：10億円→32億円、協業件数：0→5件

## 5. この会社の課題・ニーズ（推測できるもの）
- 海運は市況変動（ボラティリティ）が大きく、海運依存からの脱却（非海運拡大）に伴い、事業特性の異なる領域を横串で管理できる“共通のモノサシ（IFRS）”とデータ基盤が必要。
- 脱炭素（GHG削減）を投資・運航・顧客要請（排出量可視化、CII等）に結びつけるため、船舶データの高頻度収集、分析、意思決定への組込み（ダッシュボード/シミュレータ）が不可欠。
- 船上業務は重複記録・手入力が多く、ヒューマンエラーや負荷が安全リスクにもつながるため、センサー連携・標準化・生成AI等による省力化ニーズが高い。
- 船舶の常時接続化（通信高度化）によりOT/ITのサイバーリスクが増大し、規制適合と実運用（教育・監視・標準構成）の両面で継続強化が必要。
- DXを“現場実装”まで進めるため、全社員の変革推進力（チェンジリーダー）と、データ/AI/基盤構築を担う専門人財（国内外採用含む）の確保・育成がボトルネックになり得る。
- グローバルに分散したグループ会社・地域統括の中で、ICTガバナンス、SaaS統制、データガバナンスを効かせつつ、スピードも落とさない運営設計が必要。
