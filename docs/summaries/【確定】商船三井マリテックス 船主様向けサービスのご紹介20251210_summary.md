# 【確定】商船三井マリテックス 船主様向けサービスのご紹介20251210.pptx

## 1. ドキュメントの目的・概要（1-2文）
商船三井マリテックス（MOLMAT）が、船主向けに提供する海事ソリューション（点検整備、海事手続、新造船建造監理、海技訓練）を体系的に紹介する資料。2025年4月のグループ技術系3社統合により、建造から運航・訓練までをシームレスに支援できる体制を訴求している。

## 2. 主要なポイント（箇条書き5-10項目）
- 2025年4月に商船三井グループの海事技術系3社を統合し「商船三井マリテックス」として発足、運航・建造の知見を外部船主にも提供。
- Vision/Missionとして、オフショア領域の強みと技術力・運航知見で安全と環境保全を支え、カーボンニュートラル社会に貢献することを掲げる。
- 船主向けサービスは4本柱：①オンサイト事業（安全設備点検等）②海事手続事業（労働協約/日本籍関連/JISS等）③新造船建造監理事業（計画～竣工の一貫支援）④海技訓練事業（各種シミュレータ/DP訓練）。
- オンサイト事業では救命艇・救助艇・進水装置・離脱装置の点検整備を実施し、IMO MSC.402(96)等の要求に対応（メーカー承認・旗国認可・ISO取得）。
- ガス検知器（理研計器）点検、救命筏、消火設備、GMDSS機器等の点検手配も提供し、船舶の安全・監査対応を支援。
- 海事手続事業では、国際労使交渉に基づくGC/LC関連事務、組合対応、基金・組合費支払い代行、事故時補償手続などを代行。
- 日本籍/準日本籍船の登録・検査・証書更新、脱籍や船名/船主変更など、官庁・関係者調整を含む手続きを支援。
- 新造船建造監理は、造船所選定、仕様策定、設計レビュー、FAT立会、建造監督派遣、HSE活動までをPMが統括し、品質・工程・安全を管理（ISO9001ベース）。
- 海技訓練は操船・荷役・機関・BRM/ERMなどをシミュレータで実施し、国交省認定の操船シミュレータ講習（PEC等）や、DNV認証のLNG訓練を提供。
- DP（Dynamic Positioning）訓練は、日本初のNI認証トレーニングセンターとして、洋上風力等の特殊船向け資格取得・更新に必要な修了証書を発行。

## 3. 具体的な数字・事実（あれば箇条書き）
- 会社統合：2025年4月に3社（MOLマリン＆エンジニアリング、MOLシップテック、商船三井オーシャンエキスパート）を統合して発足。
- 従業員数：186名（2025年4月1日）。
- 株主：商船三井 100%。
- オンサイト点検実績：点検総数 2024年124隻、2025年104隻（2025年10月時点の記載あり）。
- 旗国認可等：パナマ／バハマ／リベリア主管庁認可、ClassNK、ISO9001(2015)（救命艇・進水装置点検整備関連）。
- MSC.402(96)：2020年1月1日以降、年次/5年次点検等は旗国認可整備事業所での実施が必要。
- 海事手続（GC/LC）実績：船舶管理会社17社、対象約260隻（2025年10月現在）。
- 日本籍/準日本籍関連：対象 日本籍船76隻（2025年10月現在）。
- JISS関連：対象マンニング会社13社、承認試験開催（フィリピン5回、インド2回、ブルガリア1回）。
- 新造船プロジェクト実績（2025年3月時点）：LNG燃料PCTC（6隻＋4隻）、LNGケープサイズBC（7隻＋1隻）、LNG VLCC（6隻）等に加え、メタノール運搬船1隻、液化エタン船（計6隻）、LNG運搬船（3隻＋10隻＋4隻）、13kTEUコンテナ船6隻、CSOV 2隻など。
- 訓練認証：操船シミュレータは国交省認定（PEC履歴換算の認定証明書発行）、LNG訓練はDNV GL認証、DPはNI認証（日本初）。
- 拠点/施設：本社（虎ノ門）、操船シミュレータ（芝浦）、機関・荷役シミュレータ（虎ノ門ヒルズ ステーションタワー35階）ほか、九州/広島/関西事務所。

## 5. この会社の課題・ニーズ（推測できるもの）
- 統合直後（2025年4月発足）であるため、外部船主への認知拡大とサービス横断（点検・手続・建造・訓練）の一体提案による受注拡大が重要。
- 環境規制・代替燃料・風力推進・オフショア案件の増加に伴い、設計リスク評価（HAZID等）や建造品質、HSE、運航訓練まで含む高度な支援需要が拡大。
- MSC.402(96)等の規制対応やSIRE等監査対応で、認可・品質保証（ISO）を維持しつつ、国内外（シンガポール/中国/韓国等）での手配力・供給能力を強化する必要。
- 海技者不足・経験値不足を補う訓練需要が高く、シミュレータ訓練（PEC/BRM/ERM/DP等）の提供枠拡大やプログラム高度化が求められる。
