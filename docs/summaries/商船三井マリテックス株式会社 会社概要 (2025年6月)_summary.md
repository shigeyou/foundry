# 商船三井マリテックス株式会社 会社概要 (2025年6月).pdf

## 1. ドキュメントの目的・概要（1-2文）
商船三井マリテックス株式会社（MOL Maritex Co.,Ltd.）の会社概要として、事業内容、役員体制、拠点、従業員数、沿革、業績推移、組織体制を整理した資料（2025年6月時点）です。商船三井100%子会社としての機能（海事コンサル、技術支援、訓練、船舶管理等）と、直近の統合（社名変更）後の体制が示されています。

## 2. 主要なポイント（5-10項目）
- 商船三井（MOL）100%出資のグループ会社で、海事・環境領域の調査/コンサルから船舶技術、訓練、運航・管理まで幅広く担う。  
- 定款上の事業は、海事・環境コンサル、船舶/海洋構造物の新造・改造・修繕の計画設計・監督、機器開発/保守、船員教育、船舶管理・配乗・運航、海運関連、燃料油分析、技術者育成支援、派遣、保険代理、不動産等。  
- 操船シミュレータやDP（Dynamic Positioning）シミュレータ等の訓練基盤を持ち、外部認証（ClassNK、DNV、The Nautical Institute等）を取得して訓練品質を担保。  
- 本社は虎ノ門（商船三井ビル）で、田町に操船シミュレータルームを配置。関西（神戸）・広島・九州（門司）にも事務所を展開。  
- 2025年4月に複数社統合により「商船三井マリテックス」へ社名変更し、組織もコーポレート機能、訓練、オフショア、新造船PM、オンサイト等の事業本部制で整理。  
- 役員は常勤取締役・執行役員に加え、商船三井の執行役員/部長クラスが非常勤取締役・監査役として関与し、グループガバナンス色が強い。  
- 近年の業績推移が掲載され、年度により利益の振れがある（赤字年度もあり）。  
- 訓練拠点は2024年7月に「MOL Kakio Institute」閉鎖後、虎ノ門の多目的施設「SANGO」で訓練業務を開始するなど、拠点再編が進んでいる。
