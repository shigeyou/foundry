# （CSOメッセージ）BA2035 Phase2策定にあたって.pdf

## 1. ドキュメントの目的・概要（1-2文）
「Blue Action 2035（BA2035）Phase 2」の策定に向け、Phase 1（事業領域拡大）から転換して、事業強化・収益力改善・キャッシュ創出を最重要テーマとして各部門に作成方針を示すCSOメッセージ。あわせて、財務（負債増）と組織運営（人員・費用）の背景課題を共有し、全社横断での資源配分最適化を求めている。

## 2. 主要なポイント（箇条書き5-10項目）
- Phase 2は「方向性を見定めた上での事業強化」と「収益力の点検・改善」を最重要テーマにする（Phase 1の拡大路線から転換）。
- 既存事業および既決投資案件について、収益力を点検し、改善のための体制整備と具体的アクションプランを作る。
- 事業を「勝ち筋」としてモデル化し、どのようにキャッシュを生むかを明文化・図式化する（抽象論ではなくCFに至る道筋を具体化）。
- 既存事業の継続を前提とせず、全社方針との整合と再点検を踏まえ、資産・資本回転も意識して最適な事業ポートフォリオを形成する。
- グループとして社内外に示す「全社ストーリーライン（注力方向性）」を策定し、資源配分判断の重要材料とする。
- 部門・会社を跨いだ協業（例：ケミカル物流総合戦略のような横断テーマ）を、事業モデルとして共同で検討し、価値創出につなげる。
- 成長投資を継続するために「キャッシュ化」を重視し、資産/事業の入替やパートナー招へい等、踏み込んだ計画を検討する。
- Phase 1で膨らんだ店費（人件費・コンサル費等）を踏まえ、収益力強化と同時に費用最適化・コスト競争力強化を進める。
- 事業領域拡大後の実行局面に備え、経営管理・事業開発/運営のレベルアップと、マトリックス型協業を含む連携強化を継続する。

## 3. 具体的な数字・事実（あれば箇条書き）
- 2025年度以降の数年間：既決投資の「実行」から「収益開始」までの時間差により、有利子負債の積増し余力に圧力がかかる見込み。
- 2025年9月30日 取締役会：戦略・ビジョン討議資料（「全社ストーリーラインの整理・打ち出し」）の抜粋が参考として言及されている。
- Phase 1の結果：新規事業・既存事業拡大の意思決定が「量的には期待を大幅に上回る」と記載（定量値は未記載）。

## 5. この会社の課題・ニーズ（推測できるもの）
- **負債増加と投資回収タイムラグ**により、当面の資金余力が制約されるため、既存・既決案件からのキャッシュ創出力を早期に高める必要がある。  
- **投資の選別眼（投資基準の引き上げ）**が必要で、案件の「勝ち筋」をCF起点で検証できる仕組み（評価・モニタリング）が求められている。  
- **部門最適の限界**があり、グループ横断での役割分担・統合戦略（物流等）によるシナジーの“実体化”が必要。  
- Phase 1で増えた**人件費・外部費用のコントロール**と、実行フェーズに向けた**人材再配置・業務効率化**が不可欠。  
- 社内外に示す**成長ストーリー（ストーリーライン）**を明確化し、ポートフォリオ入替やパートナー活用も含めた一貫した資源配分を行うニーズが高い。
