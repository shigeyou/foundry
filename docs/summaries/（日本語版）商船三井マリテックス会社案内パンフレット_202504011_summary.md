# （日本語版）商船三井マリテックス会社案内パンフレット_202504011.pdf

## 1. ドキュメントの目的・概要（1-2文）
商船三井マリテックス株式会社の会社概要と、提供する海事サービス（コンサル、訓練、建造監理、特殊船運航、手続・現場支援等）を紹介するパンフレットです。2025年4月1日に商船三井グループ3社の統合で発足した新会社として、DX/GXを核に「海の安全と環境保全」をリードする方針を示しています。

## 2. 主要なポイント（箇条書き5-10項目）
- 商船三井グループ内の3社（MOLマリン＆エンジニアリング、商船三井オーシャンエキスパート、MOLシップテック）が合併し、新会社として発足。  
- 社名「Maritex」は海事（MARI）×技術（TECH）×専門家（EXPERT）を志向し、「X」に未来志向・未知への挑戦、3社能力の掛け合わせ（クロス）を込める。  
- ビジョンは「海の安全と環境保全をリードする海事サービス」；デジタル活用（DX）とカーボンニュートラル対応（GX）を中核に据える。  
- 海事コンサルティング：航行環境評価、港湾機能評価、操船シミュレーションによるリスク検証、係留安全性、津波影響評価、サプライチェーン最適化、ドローン点検などを提供。  
- グローバル案件にも対応し、港湾開発・航行安全・再エネ等の国際的海事課題に関する支援を展開。  
- GX支援：洋上風力の調査〜建設〜運用支援、次世代燃料（LNG/水素/アンモニア/バイオ燃料/メタノール等）導入支援、GHG削減に向けた輸送最適化・省エネ運航支援。  
- 訓練事業：操船・機関・荷役・DP（ダイナミックポジショニング）各シミュレータを用い、BRM/ERM等の実践訓練をグループ内外へ提供。  
- シミュレータ販売：簡易構成から大規模設備まで提案し、導入後の運用支援まで一貫対応。  
- 新造船建造監理：造船所選定、仕様策定、設計確認、工程・品質管理、試運転立会、HSE活動など計画〜竣工までPM体制で支援（新燃料・DX適用も含む）。  
- 特殊船・現場支援：海底ケーブル敷設船の運航管理、研究船の運航・観測支援、オフショア/洋上風力向け特殊船のテクニカル支援、海事手続や港湾・本船でのオンサイト支援を実施。

## 5. この会社の課題・ニーズ（推測できるもの）
- **統合直後の組織融合**：3社の人材・業務プロセス・ブランドを早期に統一し、クロスセル可能な体制（営業/案件管理/品質基準）を整える必要。  
- **DX/GX案件の実装力強化**：規制・技術が急変する中で、データ活用、シミュレーション、運航最適化、代替燃料対応などを“コンサルに留めず実行”まで支える体制が求められる。  
- **訓練需要の増加への対応**：LNG燃料船や洋上風力・オフショア向けDP等の需要増に対し、講師人材・設備稼働・カリキュラム更新を継続的に拡充する必要。  
- **特殊船・研究船の安定運航**：24時間以内出航体制など高い即応性・安全性が求められ、要員確保、保守計画、リスク管理の高度化が課題になり得る。  
- **グローバル展開の体制整備**：新興国港湾や国際プロジェクト対応に向け、現地パートナー連携、多言語対応、国際規格・契約実務の強化が必要。
