# 【1月27日経営会議資料】 3Ｑ決算実績値.pdf

## 1. ドキュメントの目的・概要
FY25の3Q実績（上期＋3Q）と4Q見込みを踏まえ、通期（下期見直し予算）に対する差異を全社および事業部別に整理し、経営会議向けに業績進捗を報告する資料です。

## 2. 主要なポイント
- 全社の通期見込みは、売上高・粗利・営業利益ともに下期見直し予算を下回る見通し（売上は約3.7億円未達）。
- 一方で、営業外損益の改善により、経常利益の未達幅は営業利益より小さい。
- 販管費は「部門費」は概ね抑制傾向だが、「共通費」は予算超過となっており、利益を押し下げている。
- 事業部別では、黒字貢献している部門と、営業赤字（または大幅未達）の部門が混在している。
- 「海洋事業部」系の一部（例：オフショア船、海洋技術、訓練/シミュレータ等）は部門ごとの振れ幅が大きく、利益の安定性に課題が示唆される。
- 「新造船PM事業本部」系では、オンサイト事業部（検査サービス）が大幅な売上・粗利未達となっており、全社未達の主要因の一つになっている可能性が高い。
- いくつかの部門で原価が予算を下回り粗利が上振れしている一方、販管費（特に共通費）や売上未達で相殺されている。
- 過年度修正損の計上がある（規模は小さいが、会計上の調整事項として存在）。

## 5. この会社の課題・ニーズ（推測）
- **売上未達の主要因部門（特にオンサイト検査サービス）の立て直し**：受注・稼働・単価・案件採算のどこにギャップがあるかの特定と対策が必要。
- **共通販管費の増加要因の可視化と抑制**：予算超過（＋87,693千円）が営業利益を押し下げており、固定費構造の見直しニーズが高い。
- **部門別の収益変動（粗利・原価・稼働）の平準化**：黒字部門がある一方で赤字/未達部門が大きく、ポートフォリオ管理・リソース再配分が課題。
- **営業外損益に依存しない利益体質**：営業利益の未達を営業外で補っている構図のため、本業の収益力強化が必要。
- **予実管理の精度向上**：部門によって差異が大きく、見込み精度・原価見積・進行基準管理などの改善余地がある可能性。
