# (Offshore Unit)B. （BA2035 Phase2）事業間協業戦略(オフショア事業提出)特殊船(ケーブル船、調査研究船、その他作業船)のE2Eソリューション提供モデル(MOLMATＸ風力事業Ｘオフショア事業).pptx

## 1. ドキュメントの目的・概要（1-2文）
商船三井グループ内（オフショア事業×風力事業×MOLMAT×地域組織）の機能を統合し、特殊船（通信/電力ケーブル船、調査研究船、各種作業船）のE2E（建造〜運航・管理〜支援）ソリューションで新規案件の獲得と競争力強化を狙う協業戦略を示す。  
通信・電力ケーブル、洋上風力、オフショアO&G（石油・ガス）を横断して船隊・知見を共用し、市況リスク低減と収益力向上を目指す。

## 2. 主要なポイント（箇条書き5-10項目）
- 本社（オフショア/風力）のネットワーク・信用力・資本力、MOLMATの建造管理〜船舶管理・運航支援、地域組織（MOLAO等）の営業網を融合してE2E提供モデルを構築。  
- 新規ターゲット：通信/電力ケーブル敷設・メンテナンス船、洋上風力関連作業船、「しらせ」後継等の調査研究船の契約獲得。  
- 通信ケーブル船と電力ケーブル船は業界プレーヤーが分かれがちだが、両方に取り組む強みを活かし「新造船の共用化」や「一体管理体制」を検討し船隊競争力を強化。  
- 洋上風力とオフショアO&Gの両市場で使える作業船を整備し、需給に応じた配船で収益力強化と市況リスク低減を図る。  
- 事業間相乗効果（収益力強化・コスト低減）を狙い、部門横断で役割分担し継続的に情報交換・連携する運営モデル。  
- 役割整理：オフショア事業＝通信ケーブル船営業/事業開拓＋O&G中心の支援船、風力事業＝電力ケーブル船営業/事業開拓＋洋上風力支援船、MOLMAT＝建造/運航・船舶管理のスーパーバイズ、地域組織＝地域別の営業・事業開拓。  
- 国内：経済安全保障の観点から通信ケーブル船需要の拡大余地（潜在顧客/パートナーとしてKDDIグループ、NEC等）。電力ケーブル船は浮体式洋上風力の拡大に伴い需要増を見込む。  
- 調査研究船：「しらせ」後継は砕氷船の知見を活かせ、官公庁との関係強化が期待。  
- 海外：シンガポール（国際海底ケーブル集積）や欧州（海底電力ケーブル産業先行）を中心に、本社・MOLMAT・地域組織が連携して営業展開。  
- 既存パートナーKCSとの長年の関係を基盤に、ケーブル工事エンジニアリング知見/リソースとMOLMATの運航・管理知見を組み合わせ、多面的協力体制を構築する。

## 3. 具体的な数字・事実（あれば箇条書き）
- BA2035 Phase2の「事業間協業戦略」として位置付け。  
- 対象船種：通信/電力ケーブル敷設・メンテナンス船、洋上風力関連作業船、調査研究船（「しらせ」後継）。  
- 地域重点：APAC（シンガポール等）、欧州。

## 5. この会社の課題・ニーズ（推測できるもの）
- 通信/電力ケーブル船で分断されがちな市場構造の中で、船隊・運航管理・営業を統合して差別化し、案件獲得確度を上げる必要。  
- 洋上風力とO&Gの市況変動に備え、汎用性の高い作業船の整備と柔軟な配船で稼働率・収益の安定化が必要。  
- 建造〜運航までのE2E提供を成立させるため、社内横断の役割分担・情報連携の常態化（ガバナンス/意思決定の仕組み）が必要。  
- ケーブル工事エンジニアリング（施工側）の知見・人材・体制がボトルネックになり得るため、KCS等との協業深化が不可欠。  
- 国内では経済安全保障ニーズを踏まえた官民顧客（通信・官公庁）への提案力強化、海外では欧州での実績確保とAPAC展開の営業網強化が必要。
