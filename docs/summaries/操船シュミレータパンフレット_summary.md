# 操船シュミレータパンフレット.pdf

## 1. ドキュメントの目的・概要（1-2文）
本資料は、MOLグループ（MOLMC／MOL Maritex）の操船シミュレータ（フルミッション・ブリッジ）の特徴と活用用途（訓練・評価）を紹介するパンフレットです。特に「360°視界」と「下方視界」による自然な見張り環境を強みとして、BRM訓練や港湾計画の検証に使える点を訴求しています。

## 2. 主要なポイント（箇条書き5-10項目）
- フルミッション・ブリッジシミュレータとして、実船に近い総合的な操船環境を提供する。  
- 360°視界に加え、従来シミュレータで死角になりがちだった「後方・下方視界」を自然に観察できる。  
- 下方視界により、離着桟時の「船体と岸壁（桟橋）の位置関係」を直感的に把握しやすい。  
- 360°視界は、複数乗員がブリッジ内で相互にやり取りしながら行うBRM訓練（チーム訓練）に適する。  
- ECDIS、VHF、操舵スタンド、レーダー、ジャイロリピータ等のブリッジ機器を備える。  
- 計画中の港・新バースをシミュレータ上に再現し、風・潮流・波・浅水影響・バンク効果などを含めて操船検証できる。  
- 港内の大規模構造物（例：海上空港、橋梁）の影響を、360°表示で視認しながら評価・検討できる。  
- 人間の視認・心理（港の見え方、死角の影響）や、操縦性能／安全操船の評価にも利用できる。  
- さまざまな船種・気象海象・港湾条件で反復訓練でき、入出港の実務に近い経験を積める。  
- Bridge2にはDPS（Dynamic Positioning System）を搭載し、ケーブル船や海洋調査船等の操船模擬にも対応（通常船としてBridge1と同一海域での運用も可能）。

## 3. 具体的な数字・事実（あれば箇条書き）
- 視界：360°および下方視界（downward view）を提供。  
- 導入開始：1996年に初導入。  
- 規格対応：IMO Model Course 1.22に2014年5月に完全準拠。  
- 認証：ClassNK（日本海事協会）の認証を取得。  
- 構成：Bridge1／Bridge2の2ブリッジ構成（Bridge2はDPS搭載）。

## 5. この会社の課題・ニーズ（推測できるもの）
- 実船に近い見張り・コミュニケーション環境を再現し、BRM（チーム運航）能力を継続的に高度化したい。  
- 離着桟時の事故リスク低減のため、死角（後方・下方）を含む視界再現性を高めた訓練・評価が必要。  
- 新設バースや港湾改修、大規模構造物の建設に伴う操船安全性を、事前にシミュレーションで検証したい。  
- 多様な船種（DPS船含む）・条件で反復訓練できる設備を整え、教育品質を規格（IMO）・認証（ClassNK）に沿って担保したい。  
- 環境変化（港湾環境・運航要件・船種の多様化）に合わせ、シミュレータ／訓練体系を継続的にアップデートする必要がある。
