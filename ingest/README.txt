============================================
RAG 自動インジェスト用ディレクトリ
============================================

このディレクトリにファイルを配置すると、
次回のアプリ起動時に自動的にRAGドキュメントとして
取り込まれます。

【対応ファイル形式】
- PDF (.pdf)
- Word (.docx)
- PowerPoint (.pptx)
- テキスト (.txt)
- Markdown (.md)
- JSON (.json)
- CSV (.csv)
- メール (.msg, .eml)

【使い方】
1. このディレクトリにファイルを配置
2. アプリを起動（npm run dev）
3. 自動的にRAGに取り込まれます

【処理済みファイル】
取り込み完了後、ファイルは自動的に
「processed」サブディレクトリに移動されます。

【注意事項】
- 同じファイル名のドキュメントが既に存在する場合は
  スキップされます
- 日本語ファイル名も対応しています
